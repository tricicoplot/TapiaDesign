# TapiaDesign - Consultoría de Diseño de Interiores

Proyecto modular de desarrollo web para la evaluación del taller, enfocado en el diseño de espacios residenciales, comerciales y corporativos en Santiago de Chile.

## Estructura del Proyecto
El sitio web está dividido de forma modular utilizando componentes HTML independientes y hojas de estilo específicas por sección para mantener un código limpio y mantenible.

## Tecnologías Utilizadas
* HTML5 Semántico
* CSS3 (Flexbox, CSS Grid y Media Queries para diseño responsive)
* JavaScript nativo para la validación del formulario de contacto
